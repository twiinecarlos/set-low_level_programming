# 0. Positive anything is better than negative nothing

This program assigns a random number to the variable `n` each time it is executed.

It then checks whether the number is:
- Positive
- Zero
- Negative

## Output format

The program prints:
- The number stored in `n`
- Followed by:
  - "is positive" if n > 0
  - "is zero" if n == 0
  - "is negative" if n < 0
